추천 도서 목록

선생님들과 지인들에게 추천받은 책 목록을 공유합니다.
읽은 책의 제목은 북리뷰 포스팅으로 링크가 연결되어 있습니다.

프로그래밍 일반 및 교양 도서

  • 소프트 스킬 - 평범한 개발자의 비범한 인생전략 71가지 (울군님 추천)
    개발자의 삶과 삶의 태도, 사고방식을 다룬 자기계발서

  • CODE 코드 - 하드웨어와 소프트웨어에 숨어 있는 언어 (울군님 추천)
    컴퓨터가 어떻게 돌아가는지 본질적 원리를 설명

  • Clean Code 클린 코드 - 애자일 소프트웨어 장인 정신 (울군님 추천)
    객체지향형 프로그래밍에서 지켜야 할 원칙

  • 코딩을 지탱하는 기술 - 원리로 깨우치는 프로그래밍 기법 (름바님 추천)
    다양한 프로그래밍 언어의 특성이 어떻게 탄생했는지 알려주는 책

  • 해커와 화가 (주원님 추천)
    야후 만든 폴 그레이엄이 쓴 책. 결국 코딩을 하는 것도 이유와 논리를 가지고 있다는 것을 알려준다…고 하고 제목부터 재밌을 것 같다.

Python

  • (파이썬을 이용한) 클린 코드를 위한 테스트 주도 개발 (울군님 추천)
    장고를 하기 위해 필독할 것.

  • Two Scoops of Django 3.x (울군님 추천)
    장고 입문 전에 쉬운 이해를 위해 읽어볼 것. 2016년판 말고 온라인에 있는 3.x 버전을 위한 개정판으로 읽을 것.

  • 깐깐하게 배우는 파이썬 3 : 기본기를 다져주는 단계별 학습법 (울군님 추천)
    파이썬의 기본지식을 다질 수 있는 책

JavaScript

  • 타입스크립트 프로그래밍 - 더 빠르고 안정적인 자바스크립트 앱 개발을 위한 핵심 기능과 활용법 (울군님 추천)
    타입스크립트를 배우는 걸 추천하며 이를 위해 읽어봐야 할 책

HTML5 / CSS3 / Web Standards / Web Accessibility

  • 제프리 젤드만의 웹표준 가이드 - 웹 디자이너와 개발자, 그리고 사용자를 위한 올바른 선택 (슬비님 추천)
    웹표준에 대해 집대성한 책. 읽느라 고생했지만 읽었다는 것만으로도 기쁘고 정리한 것을 다시 보니 뿌듯하다.

  • 다양한 예제로 배우는 CSS 설계 실전 가이드 (슬비님 추천)
    CSS도 여느 다른 언어와 같이 모듈화와 재사용성을 고려한 설계가 중요하다는 것을 느끼게 해준 책. 예제와 네이밍 방법론을 상세히 설명해두어서 나중에 프로젝트 컨벤션 짜기에 굉장히 유용할 듯 하다.

  • 웹접근성과 웹표준 (슬비님 저 ebook)
    웹접근성과 웹표준에 대한 개요와 실천을 알 수 있으며, 온라인 무료열람 가능
